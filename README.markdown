Java编程怎么学：建立知识的框架，在遇到问题时逐步完善局部的知识点，不要一头扎到书本里，书本只是工具，工具而已。

java面试题的预习过程：将所有的面试题全部列出，逐步解答，解答的过程中遇到其他技术，与本题无关的，全部分类到另外一条技能树当中去。

学习的过程避免啃书。要思考和阐述，技术是练习出来的。练习是解决一个一个的问题。

学习技术，看书要迅速，大概知道书中在说什么就行，然后单元化，模块化的去实践。

专注于当下的技术，在遇到其他的技术时，将其他技术添加到下一个单元里去。如果上一个技术影响到了当前的技术，那么就将当前的技术放下，

掌握了影响这个技术的下一个技术，再回过头来继续这个技术。

最重要的是，积累、深入、专注。如果没有这些还不如放弃。

工欲善其事必先利其器，避免长时间陷入到利器过程中。

技术只是工具。
